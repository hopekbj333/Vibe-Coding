# 📦 WP 3.3: 시각 처리 훈련 (Visual Processing)

시각적 변별력과 공간 인식 능력을 훈련합니다.

---

## 💡 비개발자를 위한 전체 설명

**뭘 만드나요?**
눈으로 보는 정보를 정확하게 구별하고, 공간적 관계를 이해하는 능력을 키우는 게임들입니다.

**왜 필요한가요?**
- 글자를 읽으려면 비슷한 글자를 구별해야 함 (ㄴ vs ㄱ, b vs d)
- 글자의 방향과 위치를 정확히 인식해야 함
- 복잡한 페이지에서 원하는 정보를 찾아야 함

**일상 예시:**
- 비슷한 글자 구별: ㄴ/ㄱ, 입/잎
- 숨은그림찾기
- 퍼즐 맞추기

## 👁️ 시나리오 1: 시각 변별

### S 3.3.1: 틀린 그림 찾기 (고급)

**📋 기술적 요구사항:**
- 두 그림에서 다른 부분 5개 찾기
- 제한 시간 내 최대한 많이 찾기
- 난이도: 차이 크기, 복잡도 조절

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[화면]
┌─────────┬─────────┐
│  그림A  │  그림B  │
│  🏠🌳🐕 │  🏠🌲🐕 │
│ (원본)  │ (차이점)│
└─────────┴─────────┘
     ↓
다른 부분을 터치!
     ↓
🌳→🌲 발견! (1/5)
```

**게임화:**
- "탐정이 되어 다른 점을 찾아보세요!"
- 힌트 버튼 (돋보기 효과)
- 찾을 때마다 별 획득

**난이도:**
| 레벨 | 차이 크기 | 차이 개수 | 시간 |
|------|----------|----------|------|
| 1 | 큼 | 3개 | 60초 |
| 2 | 중간 | 4개 | 50초 |
| 3 | 작음 | 5개 | 45초 |
| 4 | 매우 작음 | 5개 | 40초 |

### S 3.3.2: 글자 방향 훈련

**📋 기술적 요구사항:**
- b/d, p/q, ㄴ/ㄱ 등 혼동 글자 집중 훈련
- 빠르게 지나가는 글자 중 목표 글자만 터치
- 게임 패턴: Go/No-Go 변형

**💡 비개발자를 위한 설명:**

**왜 필요한가요?**
- 많은 아이들이 b와 d, p와 q를 혼동함
- 한글에서도 ㄴ과 ㄱ, 6과 9를 혼동
- 반복 훈련으로 자동화 필요

**어떻게 작동하나요?**
```
[목표] "ㄱ을 찾으세요!"
     ↓
[화면] 글자가 빠르게 지나감
  ㄴ → ㄱ → ㄴ → ㄴ → ㄱ
     ↓
ㄱ이 나올 때만 터치!
     ↓
✅ 정확도 85%!
```

**훈련 글자 쌍:**
- 한글: ㄴ/ㄱ, ㄹ/ㅂ
- 숫자: 6/9, 2/5
- 영문(옵션): b/d, p/q

### S 3.3.3: 숨은 글자 찾기

**📋 기술적 요구사항:**
- 복잡한 배경에서 특정 글자/도형 찾기
- 집중력 + 시각 탐색 훈련

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[목표] "가를 모두 찾으세요!"
     ↓
[화면] 복잡한 배경
┌─────────────┐
│ 나 가 다 라 │
│ 마 바 가 사 │
│ 아 가 자 가 │
└─────────────┘
     ↓
"가" 위치를 모두 터치
     ↓
✅ 4개 중 4개 발견!
```

**난이도:**
- 쉬움: 큰 글자, 단색 배경
- 보통: 작은 글자, 패턴 배경
- 어려움: 더 작은 글자, 복잡한 배경

## 🧩 시나리오 2: 공간 인식

### S 3.3.4: 퍼즐 맞추기

**📋 기술적 요구사항:**
- 조각난 그림을 원래 위치에 배치
- 회전이 필요한 조각으로 난이도 상승

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[화면]
완성 그림(힌트): 🐱

조각들:
┌───┬───┐
│🐱머│🐱리│
├───┼───┤
│🐱몸│🐱통│
└───┴───┘

드래그하여 맞추기!
     ↓
✅ 고양이 완성!
```

**난이도:**
| 레벨 | 조각 수 | 회전 필요 |
|------|---------|----------|
| 1 | 4조각 | X |
| 2 | 6조각 | X |
| 3 | 9조각 | X |
| 4 | 9조각 | O |

---

### S 3.3.5: 도형 회전 예측

**📋 기술적 요구사항:**
- 도형을 회전하면 어떤 모양? → 선택
- 공간 지각 능력 훈련

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[화면]
원본: ▲

"오른쪽으로 한 번 돌리면?"
     ↓
[선택] ◀  ▶  ▼
     ↓
✅ ▶ 정답!
```

**회전 유형:**
- 90도 회전 (1번)
- 180도 회전 (2번)
- 270도 회전 (3번)

---

### S 3.3.6: 거울 대칭 찾기

**📋 기술적 요구사항:**
- 제시된 도형의 거울 대칭 이미지 찾기
- 좌우/상하 대칭 구분

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[화면]
원본: 🐕→ (오른쪽 보는 강아지)

"거울에 비친 모습은?"
     ↓
[선택] 🐕→  ←🐕  🐕↓
     ↓
✅ ←🐕 정답!
```

**대칭 유형:**
- 좌우 대칭 (거울)
- 상하 대칭 (물에 비친)
