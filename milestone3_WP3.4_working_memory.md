# 📦 WP 3.4: 작업 기억 훈련 (Working Memory)

정보를 일시적으로 저장하고 조작하는 능력을 훈련합니다.

---

## 💡 비개발자를 위한 전체 설명

**뭘 만드나요?**
머릿속에 정보를 잠시 저장해두고, 그걸 활용하는 능력을 키우는 게임들입니다.

**왜 필요한가요?**
- 읽으면서 앞 내용을 기억해야 뒤 내용을 이해함
- 계산할 때 중간 결과를 기억해야 함
- 선생님 지시를 기억하며 행동해야 함

**작업 기억 vs 단순 기억:**
- **단순 기억**: 그냥 외우기 (3, 7, 2)
- **작업 기억**: 외우면서 조작하기 (3, 7, 2를 거꾸로 → 2, 7, 3)

**일상 예시:**
- 암산할 때 숫자 기억하며 계산
- 여러 단계 지시 따르기
- 책 읽으면서 등장인물 기억

## 🧠 시나리오 1: 기억 폭 확장

### S 3.4.1: N-Back 게임 (아동용)

**📋 기술적 요구사항:**
- 연속으로 나오는 그림 중 "N개 전과 같으면" 터치
- 1-Back → 2-Back → 3-Back으로 난이도 상승
- 게임화: 기억력 히어로 컨셉

**💡 비개발자를 위한 설명:**

**N-Back이 뭔가요?**
- 과학적으로 검증된 작업 기억 훈련법
- "N개 전"을 기억하며 비교해야 함

**어떻게 작동하나요?**

**1-Back (쉬움):**
```
그림 순서: 🍎 → 🍌 → 🍌 → 🍇

"바로 전과 같으면 터치!"

🍎 (패스) → 🍌 (패스) → 🍌 (터치!) → 🍇 (패스)
```

**2-Back (보통):**
```
그림 순서: 🍎 → 🍌 → 🍎 → 🍇

"2개 전과 같으면 터치!"

🍎 (패스) → 🍌 (패스) → 🍎 (터치! 2개 전이 🍎) → 🍇 (패스)
```

**게임화:**
- "기억력 히어로가 되어 패턴을 찾아라!"
- 정답 시 슈퍼파워 충전 애니메이션
- 레벨업 시 히어로 업그레이드

### S 3.4.2: 카드 기억 게임

**📋 기술적 요구사항:**
- 카드 뒤집어서 짝 찾기 (메모리 게임)
- 카드 수 증가로 난이도 조절

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[화면] 카드가 뒤집혀 있음
┌───┬───┬───┬───┐
│ ? │ ? │ ? │ ? │
├───┼───┼───┼───┤
│ ? │ ? │ ? │ ? │
└───┴───┴───┴───┘
     ↓
카드 2개 터치하여 뒤집기
     ↓
🍎 과 🍎 → 짝! 사라짐
🍎 과 🍌 → 다시 뒤집어짐
     ↓
모든 짝을 찾으면 성공!
```

**난이도:**
| 레벨 | 카드 수 | 짝 수 |
|------|---------|-------|
| 1 | 8장 | 4쌍 |
| 2 | 12장 | 6쌍 |
| 3 | 16장 | 8쌍 |
| 4 | 20장 | 10쌍 |

---

### S 3.4.3: 지시 따르기 게임

**📋 기술적 요구사항:**
- 여러 단계 지시를 듣고 순서대로 수행
- "빨간 별을 터치하고, 파란 동그라미를 드래그해"

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[음성] "먼저 빨간 별을 터치하고,
       그다음 파란 동그라미를 오른쪽으로 드래그해요"
     ↓
[화면] 🔴⭐ 🔵⭐ 🔴🔵 🟢⭐
     ↓
1단계: 🔴⭐ 터치
2단계: 🔵🔵 드래그 →
     ↓
✅ 지시 완료!
```

**난이도:**
- 쉬움: 1단계 지시 ("빨간 별 터치")
- 보통: 2단계 지시 ("빨간 별 터치 후 파란 원 터치")
- 어려움: 3단계 지시 + 조건 ("빨간 별, 파란 원, 그 다음 가장 큰 것")

## 🔀 시나리오 2: 역순 기억

### S 3.4.4: 거꾸로 터치

**📋 기술적 요구사항:**
- 순서대로 불빛이 켜진 후, 거꾸로 터치
- 순방향 기억 → 역방향 변환 훈련

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[시연] 불빛 순서: 🔴 → 🔵 → 🟢
     ↓
[지시] "거꾸로 터치하세요!"
     ↓
[정답] 🟢 → 🔵 → 🔴
     ↓
✅ 역순 완벽!
```

**왜 어려운가:**
- 기억하는 것 + 뒤집어 생각하는 것
- 두 가지 인지 작업을 동시에 해야 함

**난이도:**
- 쉬움: 2개 역순
- 보통: 3개 역순
- 어려움: 4개 역순

---

### S 3.4.5: 거꾸로 말하기 훈련

**📋 기술적 요구사항:**
- 단어/숫자 듣고 거꾸로 말하기 (음성 입력)
- STT 연동 즉각 피드백

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
```
[음성] "삼, 칠, 이"
     ↓
[지시] "거꾸로 말해보세요!"
     ↓
🎤 "이, 칠, 삼"
     ↓
✅ 정답!
```

**단계:**
| 레벨 | 예시 | 정답 |
|------|------|------|
| 1 | 일, 이 | 이, 일 |
| 2 | 삼, 칠, 이 | 이, 칠, 삼 |
| 3 | 구, 사, 일, 오 | 오, 일, 사, 구 |
