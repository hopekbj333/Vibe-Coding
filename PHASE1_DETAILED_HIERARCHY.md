## 🎯 적응형 검사 알고리즘

### **검사 시작점 결정**

```python
def determine_starting_tier(child_age, previous_assessment=None):
    if previous_assessment:
        return previous_assessment.last_completed_tier + 1
    
    if child_age < 5:
        return 0  # Foundation
    elif child_age < 6:
        return 1  # Word Awareness
    elif child_age < 7:
        return 2  # Syllable
    else:
        return 3  # Onset-Rime (건너뛰고 시작)
```

### **다음 단계 진행 조건**

```python
def can_proceed_to_next_tier(tier_results):
    """
    Tier 완료 조건 확인
    """
    total_steps = len(tier_results)
    passed_steps = sum(1 for r in tier_results if r.passed)
    
    if tier_results.tier_number == 0:
        # Foundation: 70% 통과
        return passed_steps / total_steps >= 0.7
    elif tier_results.tier_number <= 3:
        # Tier 1-3: 75% 통과
        return passed_steps / total_steps >= 0.75
    elif tier_results.tier_number == 4:
        # Tier 4 (최고난이도): 60% 통과
        return passed_steps / total_steps >= 0.6
    else:
        # Advanced: 65% 통과
        return passed_steps / total_steps >= 0.65
```

### **건너뛰기 (Skip) 로직**

```python
def check_skip_possibility(tier_number, quick_test_results):
    """
    쉬운 Tier는 샘플 문제로 빠르게 확인 후 건너뛰기
    """
    if quick_test_results.accuracy > 0.9:
        # 90% 이상이면 해당 Tier 통과로 간주
        return True
    return False

# 예: Tier 1을 2개 문제로 빠르게 테스트
# → 90% 이상이면 Tier 1 건너뛰고 Tier 2로
```
