# 📦 WP 1.8: 결과 리포트 (Analysis & Reporting)

검사와 채점이 완료된 데이터를 시각화하고, 학부모/교사가 이해할 수 있는 가이드를 제공합니다.

## 📊 시나리오 1: 점수 산출 로직 (Score Calculation)

- [ ] **S 1.8.1**: 영역별 원점수 계산
    - 5대 영역별로 정답 수 / 전체 문항 수 계산
    - 하위 능력별(예: 음운 인식 1~5단계) 세부 점수 산출

- [ ] **S 1.8.2**: 가중치 적용 및 환산
    - 문항 난이도에 따른 가중치 적용 (어려운 문항 = 높은 배점)
    - 100점 만점 환산 점수 또는 백분위 산출

- [ ] **S 1.8.3**: 반응 시간 분석 (주의 집중 영역)
    - 평균 반응 시간, 표준편차 계산
    - 후반부 반응 시간 증가율 (집중력 저하 지표)

## 🚦 시나리오 2: 준비도 판정 (Readiness Classification)

- [ ] **S 1.8.4**: 신호등 알고리즘 적용
    - 영역별 점수를 기준값과 비교하여 판정:
        - 🟢 **준비 완료**: 80% 이상 → 한글 학습 시작 가능
        - 🟡 **도움 필요**: 50~79% → 앱 내 훈련 콘텐츠 권장
        - 🔴 **집중 훈련**: 50% 미만 → 전문가 상담 권유

- [ ] **S 1.8.5**: 위험 징후 감지
    - 특정 패턴 감지 시 별도 알림:
        - 1단계(청각 주의) 극저점 → 청력 검사 권유
        - 4단계(음소 조작) 심각한 결함 → 난독증 전문 평가 권유
    - 해당 메시지를 리포트에 강조 표시

## 📈 시나리오 3: 리포트 시각화 (Report Visualization)

- [ ] **S 1.8.6**: 레이더 차트 생성
    - 5대 영역 점수를 오각형 레이더 차트로 표시
    - 기준선(평균) 대비 아동의 프로필 한눈에 파악

- [ ] **S 1.8.7**: 하위 능력 막대 그래프
    - 음운 인식 1~5단계 등 세부 영역 막대 그래프
    - 색상으로 신호등 상태 표시 (초록/노랑/빨강)

- [ ] **S 1.8.8**: 강점/약점 요약 카드
    - "잘하는 영역: 시각 처리 🟢"
    - "보완이 필요한 영역: 음운 인식 3단계 🟡"
    - 텍스트 + 아이콘으로 직관적 표현

## 📝 시나리오 4: 중재 가이드 (Intervention Guide)

- [ ] **S 1.8.9**: 맞춤형 권장 사항 생성
    - 🟡 영역: "앱 내 '[음절 놀이]' 훈련을 시작해보세요"
    - 🔴 영역: "전문가 상담을 권장합니다. [가까운 센터 찾기]"
    - 🟢 영역: "훌륭해요! 다음 단계 학습을 시작해도 좋습니다"

- [ ] **S 1.8.10**: 가정 내 활동 가이드
    - 각 약점 영역에 맞는 가정 내 놀이/활동 제안
    - 예: "음절 분해가 약해요 → 박수 치며 단어 쪼개기 놀이"
    - 학부모가 바로 실천할 수 있는 구체적 팁

## 📤 시나리오 5: 리포트 공유 및 저장 (Export & Share)

- [ ] **S 1.8.11**: PDF 리포트 생성
    - 전체 결과를 1~2페이지 PDF로 내보내기
    - 차트 + 요약 + 권장사항 포함

- [ ] **S 1.8.12**: 리포트 공유
    - 카카오톡/이메일 등으로 PDF 공유 기능
    - 전문가 상담 시 제출용 자료로 활용

- [ ] **S 1.8.13**: 이력 저장 및 비교
    - 과거 검사 결과와 현재 결과 비교 뷰
    - 시간에 따른 성장 추이 그래프 (마일스톤 3에서 확장 예정)
