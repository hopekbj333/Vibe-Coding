# 📦 WP 1.5: 검사 모듈 B - 감각 처리 (Sensory Processing)

청각/순차 처리와 시각적 처리, 두 영역을 다룹니다. **터치 기반 입력**으로 자동 채점이 가능합니다.

## 👂 시나리오 1: 청각 및 순차 처리 능력 (Auditory Sequential Processing)

- [ ] **S 1.5.1**: 소리 순서 기억하기
    - 3~4개의 악기 소리(예: 북-피아노-벨)를 순차 재생
    - 화면에 악기 이미지 4개 제시 → 들은 순서대로 터치
    - 정답: 터치 순서와 재생 순서 일치 여부 비교

- [ ] **S 1.5.2**: 동물 소리 순서 맞추기
    - 동물 울음소리 시퀀스(예: 멍멍-야옹-꿀꿀) 재생
    - 동물 그림 카드를 순서대로 터치
    - 난이도 조절: 3개 → 4개 → 5개로 증가

- [ ] **S 1.5.3**: 위치 순서 기억하기
    - 화면의 여러 위치에서 순서대로 불빛이 깜빡임
    - 깜빡인 순서대로 해당 위치 터치 (Simon Says 변형)

## 👁️ 시나리오 2: 시각적 처리 능력 (Visual Processing)

- [ ] **S 1.5.4**: 다른 그림 찾기
    - 4~6개의 유사한 그림 중 하나만 다름 (예: 사과 4개 중 1개는 색이 다름)
    - 다른 그림 터치 → 정오답 자동 판정

- [ ] **S 1.5.5**: 같은 형태 찾기
    - 상단에 목표 도형(예: 별) 제시 → 하단 4개 중 동일한 형태 찾기
    - 회전/크기 변형된 보기 포함하여 난이도 조절

- [ ] **S 1.5.6**: 방향이 다른 글자 찾기
    - b/d, p/q, ㄴ/ㄱ 등 좌우/상하 대칭 글자 구별
    - 4개의 글자 중 방향이 다른 1개 찾기
    - 예: [b] [b] [d] [b] → 'd' 터치

- [ ] **S 1.5.7**: 숨은 그림 찾기
    - 복잡한 배경 안에서 특정 도형/패턴 찾기
    - 찾은 위치 터치 → 정해진 영역 내 터치 시 정답 처리
