## 🔬 학계 이론 종합 분석

### **1. 음운 인식 (Phonological Awareness) 발달 단계**

**출처:** All Ohio Early Literacy Development (2024), Gillon (2004), Goswami & Bryant (1990)

#### **발달 순서 (연령 기반)**

```
Stage 1: Word Awareness (0-2세)
   └─ 문장에서 개별 단어 구별

Stage 2: Rhyme Awareness (2-3세)
   └─ 운율/각운 인식

Stage 3: Syllable Awareness (3-4세)
   ├─ Syllable Segmentation (분해)
   ├─ Syllable Blending (합성)
   └─ Syllable Deletion (탈락)

Stage 4: Onset-Rime Awareness (5-5.5세)
   ├─ Rhyme Recognition (각운 인식)
   ├─ Rhyme Production (각운 생성)
   └─ Onset Identification (초성 식별)

Stage 5: Phoneme Awareness (5.5-6세)
   ├─ Phoneme Segmentation (음소 분해)
   ├─ Phoneme Blending (음소 합성)
   └─ Phoneme Manipulation (음소 조작: 추가/탈락/대치)
```

**총 단계: 5단계**  
**세부 능력: 12개**

### **2. 병렬적 인지 영역 (Pre-literacy Cognitive Domains)**

**출처:** Research-based Pre-literacy Skills Analysis (2024)

#### **Domain 1: Visual Processing (시각 처리) - 7개 구인**

| # | 구인 | 설명 | 평가 방법 |
|---|------|------|----------|
| V1 | Visual Discrimination | 형태/글자 차이 인식 | 다른 글자 찾기 |
| V2 | Visual Sequencing | 이미지 순서 이해 | 순서 배열하기 |
| V3 | Visual Spatial Processing | 공간 관계 인식 | 위치 판단하기 |
| V4 | Visual Memory | 시각 정보 기억 | 본 것 기억하기 |
| V5 | Visual Closure | 불완전한 형태 완성 | 부분으로 전체 추측 |
| V6 | Visual Synthesis | 요소 통합 | 조각 맞추기 |
| V7 | Visual Tracking | 선 따라가기 | 움직임 추적 |

## 📊 **최종 능력 체계 (학계 이론 기반)**

### **선형적 발달 (Sequential Development)**

**음운 인식 (Phonological Awareness) - 5단계 12개 능력**

```
[Foundation] 0. 기초 청각 능력 (선행 조건)
   0.1 환경음 식별
   0.2 소리 크기/높이/방향 구별

[Stage 1] 단어 인식 (Word Awareness) - 2개
   1.1 문장에서 단어 구별
   1.2 단어 수 세기

[Stage 2] 운율 인식 (Rhyme Awareness) - 2개
   2.1 각운 인식 (끝소리 같은 것)
   2.2 운율 패턴 (리듬)

[Stage 3] 음절 인식 (Syllable Awareness) - 3개
   3.1 음절 분해 (Segmentation)
   3.2 음절 합성 (Blending)
   3.3 음절 탈락 (Deletion)

[Stage 4] 초성-운 인식 (Onset-Rime) - 3개
   4.1 각운 인식/생성
   4.2 초성 식별
   4.3 초성-운 분리

[Stage 5] 음소 인식 (Phoneme Awareness) - 3개
   5.1 음소 분해 (Segmentation)
   5.2 음소 합성 (Blending)
   5.3 음소 조작 (Manipulation)
```

**총: 15개 능력 (Foundation 포함 17개)**
