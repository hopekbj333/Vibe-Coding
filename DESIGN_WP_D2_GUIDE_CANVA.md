## 🎨 배지 디자인 가이드

### 디자인 원칙
- **크기**: 256x256 px (정사각형)
- **스타일**: 플랫 디자인, 밝고 화려하게
- **색상**: 각 배지별 고유 색상
- **아이콘**: 중앙에 명확한 심볼
- **배경**: 투명 (PNG)

### 색상 가이드

```
Bronze  : #CD7F32 (구리색) + 갈색 계열
Silver  : #C0C0C0 (은색) + 회색 계열
Gold    : #FFD700 (금색) + 노란색 계열

기타 배지:
- First Complete : 파란색 (#4A90E2)
- Perfect       : 초록색 (#52C41A)
- Speed         : 빨간색 (#FF4D4F)
- Streak        : 주황색 (#FA8C16)
- Master        : 보라색 (#9254DE)
- Champion      : 무지개색 또는 금색
```

## 🚀 Canva로 배지 제작 (빠른 방법)

### Step 1: 새 디자인 생성

1. 🌐 https://www.canva.com/ 접속
2. **"디자인 만들기"** → **"맞춤 크기"**
3. **256 x 256 px** 입력
4. **"새 디자인 만들기"** 클릭

---

### Step 2: 템플릿 검색 (추천)

1. 왼쪽 **"템플릿"** 클릭
2. 검색어 입력:
   - **"badge"**
   - **"medal"**
   - **"award"**
   - **"achievement"**
3. 필터: **"뱃지"**, **"아이콘"**
4. 마음에 드는 디자인 선택

---

### Step 3: 배지별 제작 가이드

#### 🥉 1. Bronze (브론즈)

**템플릿 검색**: "bronze medal"

**커스터마이징:**
```
1. 배지 템플릿 선택
2. 색상 변경: #CD7F32 (구리색)
3. 중앙 아이콘: 별 또는 방패
4. 텍스트 제거 (아동용이므로)
5. 배경 제거
```

**또는 직접 제작:**
```
1. 왼쪽 "요소" → "도형" 선택
2. 원형 선택 → 크기 200x200 px
3. 색상: #CD7F32 (그라데이션 적용)
4. "요소" → "아이콘" → "별" 추가
5. 테두리 추가 (2px, 어두운 갈색)
```

---

## 🎨 일관성 있는 배지 만들기

### 템플릿 하나로 9개 만들기 (추천)

## 📥 다운로드 및 저장

### 배경 제거

**각 배지마다:**
1. 배지 이미지 선택
2. 상단 **"배경 지우기"** 클릭
3. 투명 배경 완성

### 일괄 다운로드

1. 우측 상단 **"공유"** → **"다운로드"**
2. 파일 형식: **PNG**
3. ✅ **"투명 배경"** 체크
4. **"모든 페이지"** 선택
5. **"다운로드"** 클릭

### 파일명 변경

다운로드한 ZIP 압축 해제 후 파일명 변경:

```
페이지 1 → badge_bronze.png
페이지 2 → badge_silver.png
페이지 3 → badge_gold.png
페이지 4 → badge_first_complete.png
페이지 5 → badge_perfect.png
페이지 6 → badge_speed.png
페이지 7 → badge_streak.png
페이지 8 → badge_master.png
```

## 🆘 문제 해결

### Q1: 배지가 너무 단순해 보여요
**A**: 
- 그라데이션 효과 추가
- 테두리에 별 또는 장식 추가
- "effects" → "shadows" 적용

### Q2: 아이콘을 어떻게 찾아요?
**A**: 
- 왼쪽 "요소" → "아이콘" 검색
- 검색어: "star", "crown", "trophy", "fire", "lightning"

### Q3: 9개 배지가 다 달라 보여요
**A**: 
- 첫 배지(Bronze)를 복제해서 만들기
- 색상과 중앙 아이콘만 변경
- 크기, 테두리, 그림자는 동일하게 유지

### Q4: 배경 제거가 안 돼요
**A**: 
- 배지를 그룹화(Ctrl+G) 후 배경 제거
- 또는 처음부터 투명 배경에서 작업
