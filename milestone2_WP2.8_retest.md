# 📦 WP 2.8: 재검사 시스템 (Re-test / Mini Assessment)

학습 효과를 측정하는 미니 테스트 시스템입니다.

---

## 💡 비개발자를 위한 전체 설명

**뭘 만드나요?**
학습을 한 후에 "정말 실력이 늘었는가?"를 확인하는 짧은 테스트입니다. 마치 학교에서 배운 후 쪽지 시험을 보는 것과 같습니다.

**왜 필요한가요?**
- 게임만 하고 끝나면 실력이 늘었는지 알 수 없습니다
- 테스트로 확인해야 "진짜 배웠구나!" 확신할 수 있습니다
- 부모님도 아이의 성장을 눈으로 볼 수 있습니다

**검사 vs 미니 테스트:**
- **Milestone 1 검사**: 41문항, 30~40분 소요, 처음에 1번
- **미니 테스트**: 5~10문항, 5분 소요, 학습 후 수시로

---

## 📝 시나리오 1: 미니 테스트

### S 2.8.1: 학습 영역별 미니 테스트 생성

**📋 기술적 요구사항:**
- 각 단계 학습 완료 후 5~10문항 짧은 테스트
- 본 검사와 동일한 형식, 축소된 분량

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
1. "음절 쪼개기" 게임을 5일 동안 연습
2. 5일째 끝나면: "미니 테스트를 해볼까?" 제안
3. 5문제만 풀어봄 (5분 소요)
4. 결과 확인: "80점! 잘 배웠어!"

**문제 구성:**
- 본 검사와 똑같은 유형
- 다만 문항 수가 적음 (5~10개)
- 빠르게 실력을 확인하는 용도

---

### S 2.8.2: 미니 테스트 결과 산출

**📋 기술적 요구사항:**
- 해당 영역 점수만 재계산
- 이전 검사 결과와 비교 (향상도 표시)

**💡 비개발자를 위한 설명:**

**무엇을 보여주나요?**
```
[미니 테스트 결과]

음절 쪼개기 영역
📊 이전 점수: 50점 (🟡)
📈 현재 점수: 80점 (🟢)

🎉 30점 향상!
```

- 이전 검사 점수와 비교합니다
- 얼마나 늘었는지 보여줍니다
- 신호등 색깔이 바뀝니다 (🟡→🟢)

---

## 📈 시나리오 2: 향상도 시각화

### S 2.8.3: Before/After 비교

**📋 기술적 요구사항:**
- 막대 그래프로 이전 점수 vs 현재 점수 비교
- 향상된 점수는 초록색 상승 화살표로 강조

**💡 비개발자를 위한 설명:**

**화면 예시:**
```
음절 쪼개기
          
100% |              ████ (80점) 🟢 ↑30점!
 80% |              
 60% |     
 40% | ████ (50점) 🟡
  0% |____________________
     처음   →   2주 후
```

- 왼쪽 막대: 처음 검사 점수
- 오른쪽 막대: 2주 학습 후 점수
- 초록 화살표: 얼마나 늘었는지

**부모님이 보시기에:**
- "우리 아이가 정말 실력이 늘었구나!" 확신
- 숫자로, 그래프로 보이니 뿌듯함

---

### S 2.8.4: 성장 리포트 생성

**📋 기술적 요구사항:**
- "3주간 음절 분해 능력이 30% 향상되었어요!"
- 부모/교사용 짧은 성장 리포트

**💡 비개발자를 위한 설명:**

**무엇을 보여주나요?**
```
[철수의 3주 성장 리포트]

🎯 집중 연습한 영역
   음절 쪼개기: 50점 → 80점 (+30점)

💪 함께 향상된 영역
   음절 합치기: 60점 → 75점 (+15점)

⏰ 학습 시간
   총 7회, 105분 (하루 평균 15분)

🎮 가장 좋아한 게임
   음절 블록 쪼개기 (12회 플레이)

📝 선생님 한마디
   꾸준히 잘 하고 있어요! 계속 이렇게만 하면
   한글 배우기가 쉬울 거예요.
```

## 🏆 시나리오 3: 단계 승급 판정

### S 2.8.5: 다음 단계 진입 조건

**📋 기술적 요구사항:**
- 미니 테스트 점수가 기준(예: 80%) 이상 시 다음 단계 해제
- 미달 시 추가 학습 권장 + 재도전 기회

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**

**Case 1: 통과 (80점 이상)**
```
미니 테스트: 85점
  ↓
"축하해! 다음 단계로 갈 수 있어!"
  ↓
🔓 2단계 잠금 해제
  ↓
화려한 축하 애니메이션
```

**Case 2: 재도전 (80점 미만)**
```
미니 테스트: 70점
  ↓
"조금만 더 연습하면 돼!"
  ↓
추가 학습 권장 (3일 더)
  ↓
3일 후 다시 미니 테스트 기회
```

**왜 이렇게 하나요?**
- 완전히 배우지 못한 상태로 넘어가면 다음 단계가 너무 어렵습니다
- 확실히 배운 후 다음으로 가야 자신감이 생깁니다
- 재도전 기회를 줘서 좌절하지 않게 합니다

**부모 설정:**
- 승급 기준을 70%~90%로 조절 가능
- "우리 아이는 90% 이상일 때 다음 단계로!"
