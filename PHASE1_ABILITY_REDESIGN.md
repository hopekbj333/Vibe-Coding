## 📚 학계 이론 요약

### **음운 인식 (Phonological Awareness) 위계**

**출처:** Oregon DOE, Newcastle University

```
Level 1: Word Awareness (단어 인식)
   ↓
Level 2: Syllable Awareness (음절 인식)
   ↓
Level 3: Onset-Rime Awareness (초성-운 인식)
   ↓
Level 4: Phoneme Awareness (음소 인식)
```

**특징:**
- 큰 단위 → 작은 단위 순서
- 각 단계는 이전 단계 습득 필요 (선형적)
- 건너뛰기 어려움

### **음운 인식 선행 조건 (Prerequisites)**

**출처:** Scribd - Skill Sequences Early Literacy

음운 인식 전에 필요한 기초 능력:

1. **청각적 주의 (Listening & Attention)**
2. **환경음 인식 (Environmental Sound Awareness)**
3. **언어 리듬 (Rhythm of Language)**
4. **순서화 (Sequencing Events)**
5. **어휘 발달 (Vocabulary Development)**

### **문해력 예측 요인 (Literacy Predictors)**

**출처:** PubMed, Springer 연구

**병렬적 요인 (동시에 필요):**
- 시각 처리 (Visual Processing)
- 청각 처리 (Auditory Processing)
- 작업 기억 (Working Memory)
- 주의력 (Attention)

**관계:**
- 음운 인식 ← 청각 처리 + 작업 기억
- 읽기 이해 ← 음운 인식 + 시각 처리 + 작업 기억
- 즉, **병렬적으로 발달 필요**

## 🏗️ 재설계 구조 (안)

### **Tier 0: 기초 감각 능력 (Foundation)**

**병렬적 (동시 발달)**

```
┌─────────────────┬─────────────────┬─────────────────┐
│  청각 기초       │   시각 기초      │   인지 기초      │
│  (Auditory Base) │ (Visual Base)   │ (Cognitive Base) │
├─────────────────┼─────────────────┼─────────────────┤
│ - 환경음 식별    │ - 색 구별       │ - 주의 집중      │
│ - 소리 크기 구별 │ - 형태 구별     │ - 단순 기억      │
│ - 소리 방향      │ - 위치 인식     │ - 순서화         │
└─────────────────┴─────────────────┴─────────────────┘
```

### **Tier 1: 음운 인식 Level 1 (Word Awareness)**

**선행 조건:** Tier 0 완료  
**목표:** 단어를 소리 단위로 인식

```
Level 1.1: 단어 경계 인식
Level 1.2: 단어 수 세기
Level 1.3: 단어 분리/합성
```

### **Tier 2: 음운 인식 Level 2 (Syllable Awareness)**

**선행 조건:** Tier 1 완료  
**목표:** 음절 단위 조작

```
Level 2.1: 음절 인식 (박수)
Level 2.2: 음절 분해
Level 2.3: 음절 합성
Level 2.4: 음절 조작 (탈락, 대치)
```

### **Tier 3: 음운 인식 Level 3 (Onset-Rime)**

**선행 조건:** Tier 2 완료  
**목표:** 초성-운 조작

```
Level 3.1: 각운 인식
Level 3.2: 두운 인식
Level 3.3: 초성-운 분리
Level 3.4: 초성-운 합성
```

### **Tier 4: 음운 인식 Level 4 (Phoneme Awareness)**

**선행 조건:** Tier 3 완료  
**목표:** 음소 단위 조작

```
Level 4.1: 음소 인식
Level 4.2: 음소 분리
Level 4.3: 음소 합성
Level 4.4: 음소 조작 (탈락, 대치, 추가)
```

### **병렬 발달: 지원 능력**

**Tier 0~4 전체에 걸쳐 동시 발달 필요**

```
┌──────────────────────────────────────────────┐
│         청각/순차 처리 (Auditory)             │
│  - 소리 순서 기억                             │
│  - 리듬 패턴                                  │
│  - 음 높이/크기 변별                          │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│         시각 처리 (Visual)                    │
│  - 글자 방향 구별 (b/d)                       │
│  - 형태 인식                                  │
│  - 시각 추적                                  │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│         작업 기억 (Working Memory)            │
│  - 숫자/단어 폭                               │
│  - 거꾸로 기억                                │
│  - 이중 과제                                  │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│         주의력 (Attention)                    │
│  - 선택적 주의                                │
│  - 지속적 주의                                │
│  - 분할 주의                                  │
└──────────────────────────────────────────────┘
```
