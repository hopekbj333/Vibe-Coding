## 📦 WP 3.1: 음운 인식 4~5단계 훈련 (Phoneme & Memory)

마일스톤 2의 1~3단계에 이어 고급 음운 인식 훈련입니다.

### 🔬 시나리오 1: 4단계 - 음소 조작 (Phoneme Manipulation)

- **S 3.1.1**: 초성 분리 게임
    - "강아지의 첫 소리는?" → ㄱ/ㄴ/ㄷ 중 선택
    - 시각적 힌트: 글자에서 초성이 분리되는 애니메이션
    - 게임 패턴: 이선다지

- **S 3.1.2**: 음소 합성 게임
    - ㄱ + ㅏ = ? → 가/나/다 그림 중 선택
    - 블록이 합쳐지는 애니메이션
    - 난이도: 초성+중성 → 초성+중성+종성

- **S 3.1.3**: 음소 탈락 게임
    - "공에서 ㄱ을 빼면?" → 옹/공/강 중 선택
    - 초성/중성/종성 탈락 연습

- **S 3.1.4**: 음소 대치 게임
    - "강의 ㄱ을 ㅂ으로 바꾸면?" → 방
    - 블록 교체 드래그 인터랙션

- **S 3.1.5**: 음소 추가 게임
    - "아에 ㄱ을 붙이면?" → 가
    - 빈 블록에 음소 드래그하여 완성

### 🧠 시나리오 2: 5단계 - 음운 기억 (Phonological Memory)

- **S 3.1.6**: 비단어 따라하기 훈련
    - 존재하지 않는 단어(예: "두파리", "삐꾸롱") 듣고 따라 말하기
    - 게임화: 외계어 통역사 컨셉
    - 난이도: 2음절 → 3음절 → 4음절

- **S 3.1.7**: 숫자 폭 확장 게임
    - 숫자 시퀀스 듣고 순서대로 터치
    - 점점 길어지는 시퀀스로 기억 폭 훈련
    - 최고 기록 도전 요소 추가

- **S 3.1.8**: 단어 폭 확장 게임
    - 단어 시퀀스 듣고 해당 그림 순서대로 터치
    - 스토리텔링: "마트에서 사야 할 것들"

## 📦 WP 3.2: 청각/순차 처리 훈련 (Auditory Sequential)

소리의 순서와 패턴을 기억하고 재현하는 훈련입니다.

### 🎵 시나리오 1: 소리 순서 기억

- **S 3.2.1**: 악기 순서 게임
    - 3~5개 악기 소리 순서대로 재생 → 순서대로 터치
    - 게임화: 오케스트라 지휘자 컨셉
    - 난이도: 악기 수 증가, 유사한 소리 추가

- **S 3.2.2**: 동물 소리 이야기
    - 동물 소리 시퀀스로 짧은 이야기 구성
    - "먼저 강아지가 짖고, 그다음 고양이가..." → 순서 재현

- **S 3.2.3**: 사이먼 세즈 (Simon Says)
    - 화면 4분할, 순서대로 불빛 + 소리
    - 같은 순서로 터치하여 재현
    - 클래식 게임의 검증된 훈련 효과

### 🔄 시나리오 2: 패턴 인식

- **S 3.2.4**: 리듬 패턴 완성
    - 반복되는 리듬 패턴 중 빠진 부분 채우기
    - 예: 쿵-짝-쿵-? → 짝

- **S 3.2.5**: 소리 규칙 찾기
    - 소리 시퀀스의 규칙 파악 후 다음 소리 예측
    - 예: 높음-낮음-높음-? → 낮음

## 📦 WP 3.3: 시각 처리 훈련 (Visual Processing)

시각적 변별력과 공간 인식 능력을 훈련합니다.

### 👁️ 시나리오 1: 시각 변별

- **S 3.3.1**: 틀린 그림 찾기 (고급)
    - 두 그림에서 다른 부분 5개 찾기
    - 제한 시간 내 최대한 많이 찾기
    - 난이도: 차이 크기, 복잡도 조절

- **S 3.3.2**: 글자 방향 훈련
    - b/d, p/q, ㄴ/ㄱ 등 혼동 글자 집중 훈련
    - 빠르게 지나가는 글자 중 목표 글자만 터치
    - 게임 패턴: Go/No-Go 변형

- **S 3.3.3**: 숨은 글자 찾기
    - 복잡한 배경에서 특정 글자/도형 찾기
    - 집중력 + 시각 탐색 훈련

### 🧩 시나리오 2: 공간 인식

- **S 3.3.4**: 퍼즐 맞추기
    - 조각난 그림을 원래 위치에 배치
    - 회전이 필요한 조각으로 난이도 상승

- **S 3.3.5**: 도형 회전 예측
    - 도형을 회전하면 어떤 모양? → 선택
    - 공간 지각 능력 훈련

- **S 3.3.6**: 거울 대칭 찾기
    - 제시된 도형의 거울 대칭 이미지 찾기
    - 좌우/상하 대칭 구분

## 📦 WP 3.4: 작업 기억 훈련 (Working Memory)

정보를 일시적으로 저장하고 조작하는 능력을 훈련합니다.

### 🧠 시나리오 1: 기억 폭 확장

- **S 3.4.1**: N-Back 게임 (아동용)
    - 연속으로 나오는 그림 중 "2개 전과 같으면" 터치
    - 1-Back → 2-Back → 3-Back으로 난이도 상승
    - 게임화: 기억력 히어로 컨셉

- **S 3.4.2**: 카드 기억 게임
    - 카드 뒤집어서 짝 찾기 (메모리 게임)
    - 카드 수 증가로 난이도 조절

- **S 3.4.3**: 지시 따르기 게임
    - 여러 단계 지시를 듣고 순서대로 수행
    - "빨간 별을 터치하고, 파란 동그라미를 드래그해"

### 🔀 시나리오 2: 역순 기억

- **S 3.4.4**: 거꾸로 터치
    - 순서대로 불빛이 켜진 후, 거꾸로 터치
    - 순방향 기억 → 역방향 변환 훈련

- **S 3.4.5**: 거꾸로 말하기 훈련
    - 단어/숫자 듣고 거꾸로 말하기 (음성 입력)
    - STT 연동 즉각 피드백

## 📦 WP 3.5: 주의 집중 훈련 (Attention Training)

선택적 주의력과 지속적 집중력을 훈련합니다.

### 🎯 시나리오 1: 선택적 주의

- **S 3.5.1**: 목표물 사냥
    - 여러 물체가 움직이는 화면에서 목표물만 터치
    - 방해 자극 증가로 난이도 상승
    - 게임화: 곤충 채집, 별 따기 등

- **S 3.5.2**: 색깔-단어 게임 (스트룹 변형)
    - 아동용 단순화: 색깔과 그림이 일치하면 O
    - 예: 빨간색으로 칠해진 사과 → O, 파란색 사과 → X

- **S 3.5.3**: 청각 선택 주의
    - 여러 소리가 겹쳐 나올 때 특정 소리만 찾기
    - 예: 배경 음악 중 "딩" 소리가 나면 터치

### ⏱️ 시나리오 2: 지속적 주의

- **S 3.5.4**: 집중력 마라톤
    - 2~3분간 단순 과제 지속 수행
    - 특정 도형이 나올 때만 터치 (CPT 변형)
    - 시간별 정확도 추적으로 집중력 패턴 분석

- **S 3.5.5**: 흐름 따라가기
    - 천천히 움직이는 목표물을 계속 따라가며 터치
    - 눈을 떼면 점수 감소
    - 시선 추적 대용 훈련

## 📦 WP 3.6: 장기 추적 시스템 (Long-term Tracking)

아동의 성장을 장기적으로 추적하고 시각화합니다.

### 📈 시나리오 1: 성장 이력 관리

- **S 3.6.1**: 검사 이력 타임라인
    - 모든 검사/재검사 결과를 시간순 타임라인으로 표시
    - 특정 시점 클릭 시 해당 리포트 조회

- **S 3.6.2**: 영역별 성장 그래프
    - 5대 영역별 점수 변화를 선 그래프로 시각화
    - 기간 필터 (1개월/3개월/6개월/1년)

- **S 3.6.3**: 학습 활동 로그
    - 일별 학습 시간, 완료한 게임, 정답률 기록
    - 캘린더 뷰로 학습 패턴 확인

### 🏆 시나리오 2: 성취 시스템

- **S 3.6.4**: 배지/업적 시스템
    - 특정 목표 달성 시 배지 부여
    - 예: "7일 연속 학습", "음절 마스터", "집중력 챔피언"
    - 컬렉션 형태로 동기 부여

- **S 3.6.5**: 레벨 시스템
    - 학습량/성취에 따른 레벨업
    - 캐릭터 성장 또는 아이템 해금으로 게임화

### 🔔 시나리오 3: 알림 및 리마인더

- **S 3.6.6**: 학습 리마인더
    - 설정한 시간에 학습 알림 푸시
    - "오늘 아직 학습하지 않았어요!"

- **S 3.6.7**: 성장 알림
    - 의미 있는 성장 발생 시 알림
    - "음절 분해 능력이 20% 향상되었어요!"

- **S 3.6.8**: 재검사 권장 알림
    - 일정 학습량 도달 시 재검사 권장
    - "한 달간 열심히 했네요! 실력을 확인해볼까요?"

## 📦 WP 3.7: 결제 및 구독 시스템 (Payment & Subscription)

수익화를 위한 결제 시스템입니다.

### 💳 시나리오 1: 결제 연동

- **S 3.7.1**: 인앱 결제 연동
    - Google Play / App Store 인앱 결제 연동
    - 결제 검증 서버 로직 구현

- **S 3.7.2**: 결제 수단 관리
    - 등록된 결제 수단 조회/변경/삭제
    - 결제 실패 시 재시도 로직

### 📦 시나리오 2: 상품 구조

- **S 3.7.3**: 검사권 상품
    - 기초 검사 1회권, 5회권, 10회권

- **S 3.7.4**: 학습 구독 상품
    - 월간/연간 구독 (학습 콘텐츠 무제한)
    - 구독 시 검사권 일부 포함 혜택

- **S 3.7.5**: 무료 체험
    - 첫 검사 1회 무료 + 학습 3일 무료 체험
    - 체험 종료 후 구독 유도 화면

### 📋 시나리오 3: 구독 관리

- **S 3.7.6**: 구독 상태 관리
    - 현재 구독 상태, 갱신일, 남은 검사권 표시
    - 구독 해지/재개 기능

- **S 3.7.7**: 영수증 및 내역
    - 결제 내역 조회
    - 영수증 이메일 발송

## 📦 WP 3.8: 오프라인 지원 및 최적화 (Offline & Optimization)

네트워크 없이도 학습 가능하도록 최적화합니다.

### 📴 시나리오 1: 오프라인 모드

- **S 3.8.1**: 스마트 캐싱
    - 자주 사용하는 학습 콘텐츠 자동 다운로드
    - Wi-Fi 연결 시에만 대용량 에셋 다운로드

- **S 3.8.2**: 오프라인 학습 지원
    - 네트워크 없이 캐싱된 콘텐츠로 학습 진행
    - 학습 데이터는 로컬 저장 후 온라인 시 동기화

- **S 3.8.3**: 동기화 관리
    - 오프라인 중 쌓인 데이터 자동 동기화
    - 충돌 발생 시 서버 데이터 우선 또는 사용자 선택

### ⚡ 시나리오 2: 성능 최적화

- **S 3.8.4**: 앱 시작 속도 최적화
    - 스플래시 화면 중 필수 데이터만 로딩
    - 지연 로딩으로 초기 로딩 시간 단축

- **S 3.8.5**: 메모리 관리
    - 사용하지 않는 에셋 자동 해제
    - 저사양 기기 대응 (이미지 품질 자동 조절)

- **S 3.8.6**: 배터리 최적화
    - 백그라운드 동작 최소화
    - 게임 루프 최적화로 배터리 소모 절감

### 🔒 시나리오 3: 데이터 보안 강화

- **S 3.8.7**: 로컬 데이터 암호화
    - 민감 데이터(아동 정보, 검사 결과) 암호화 저장
    - 앱 삭제 시 데이터 완전 삭제

- **S 3.8.8**: 개인정보 관리
    - 데이터 내보내기/삭제 요청 기능
    - GDPR/개인정보보호법 준수
