## 🚀 방법 1: Canva AI (Magic Media) 사용 (추천)

### Step 1: Canva 접속 및 새 디자인 생성

1. 🌐 https://www.canva.com/ 접속
2. **"디자인 만들기"** 클릭
3. **"맞춤 크기"** 선택
4. **1024 x 1024 px** 입력
5. **"새 디자인 만들기"** 클릭

---

### Step 2: Magic Media (AI 이미지 생성) 사용

1. 왼쪽 메뉴에서 **"앱"** 클릭
2. **"Magic Media"** 검색 및 선택
3. **"Text to Image"** 탭 선택
4. 프롬프트 입력 (아래 참고)
5. **"이미지 생성"** 클릭
6. 마음에 드는 이미지 선택하여 캔버스에 추가

---

### Step 3: 캐릭터별 프롬프트 (한국어 또는 영어)

#### 1️⃣ Neutral (중립) - 기본 스타일 확정

**한국어 프롬프트:**
```
귀여운 곰 캐릭터 마스코트, 만 6세 아동용 교육 앱, 
플랫 디자인, 카툰 스타일, 중성적, 밝은 갈색 털,
정면 모습, 서 있는 자세, 부드러운 미소, 
큰 눈, 친근한 표정, 흰색 배경, 
듀오링고 스타일, PBS Kids 캐릭터 스타일
```

**영어 프롬프트:**
```
Cute friendly bear character mascot for children's literacy app, 
flat design, cartoon style, gender-neutral, warm brown fur,
front view, standing upright, gentle smile, 
large expressive eyes, friendly face, white background,
Duolingo style, PBS Kids character style, simple and clean
```

---

#### 2️⃣ Happy (기쁨)

**한국어:**
```
귀여운 곰 캐릭터, 매우 행복한 표정, 활짝 웃는 얼굴,
초승달 모양 눈 (^_^), 팔을 들어 환호하는 자세,
주변에 작은 별이나 반짝임 효과,
밝고 따뜻한 색상, 플랫 디자인, 흰색 배경
```

**영어:**
```
Cute bear character with very happy expression, big smile,
crescent moon eyes (^_^), arms raised in celebration,
small sparkles or stars around, bright warm colors,
flat design, white background
```

---

#### 3️⃣ Thinking (생각)

**한국어:**
```
귀여운 곰 캐릭터, 생각하는 표정, 손을 턱에 대고,
또는 고개를 갸웃 기울임, 눈은 위를 바라봄,
머리 위에 작은 물음표(?), 호기심 가득한 얼굴,
플랫 디자인, 흰색 배경
```

**영어:**
```
Cute bear character with thinking expression, hand on chin,
or head tilted to side, eyes looking upward,
small question mark (?) above head, curious face,
flat design, white background
```

---

#### 4️⃣ Sad (슬픔) - **중요: 너무 슬프지 않게!**

**한국어:**
```
귀여운 곰 캐릭터, 살짝 슬픈 표정 (너무 슬프지 않게),
위로하는 제스처로 팔을 앞으로 뻗음, "괜찮아"라고 말하는 듯,
눈은 약간 처진 모양이지만 여전히 따뜻함,
부드러운 색상, 격려하고 지지하는 느낌,
플랫 디자인, 흰색 배경
```

**영어:**
```
Cute bear character with slightly sad but comforting expression (NOT too sad),
arms extended forward in comforting gesture, as if saying "it's okay",
eyes slightly downturned but still warm and friendly,
soft gentle colors, supportive and encouraging feel,
flat design, white background
```

---

#### 5️⃣ Excited (신남)

**한국어:**
```
귀여운 곰 캐릭터, 매우 신나고 흥분한 표정,
반짝이는 눈, 점프하거나 팔을 활짝 벌린 자세,
주변에 불꽃놀이, 색종이, 무지개, 별빛 효과,
매우 밝고 생생한 색상, 플랫 디자인, 흰색 배경
```

**영어:**
```
Cute bear character with very excited expression,
sparkling eyes, jumping with joy or arms spread wide,
fireworks, confetti, rainbow, or star sparkles around,
very bright vibrant colors, flat design, white background
```

---

### Step 4: 페이지 복사로 일관성 유지

1. 첫 번째 캐릭터(Neutral) 생성 후 마음에 들면
2. 페이지 하단에서 **"페이지 복제"** 클릭
3. 복제된 페이지에서 캐릭터 이미지만 삭제
4. 같은 프롬프트에 감정만 바꿔서 새 이미지 생성
5. 5개 페이지 모두 제작

**팁**: 같은 세션에서 작업하면 AI가 스타일을 학습해 일관성 유지!

## 🚀 방법 2: Canva 템플릿 활용 (더 빠름)

### Step 1: 캐릭터 템플릿 검색

1. Canva 홈에서 **"템플릿"** 검색
2. 검색어: **"mascot"**, **"character"**, **"bear character"**
3. 필터: **"일러스트"**, **"캐릭터"**
4. 마음에 드는 캐릭터 선택

### Step 2: 템플릿 커스터마이징

1. 선택한 템플릿 열기
2. **색상 변경**: 상단 도구바에서 색상 팔레트 선택
3. **표정 수정**: 눈, 입 요소를 선택하여 크기/각도 조절
4. **포즈 수정**: 팔, 다리 요소를 이동/회전

### Step 3: 감정별 변형 생성

1. **Neutral** 먼저 완성
2. **페이지 복제** (Ctrl+D 또는 하단 메뉴)
3. 복제된 페이지에서 표정만 수정:
   - **Happy**: 입을 더 크게, 눈을 초승달 모양으로
   - **Thinking**: 눈을 위로, 물음표 추가
   - **Sad**: 눈과 입을 약간 아래로
   - **Excited**: 입을 크게, 별/반짝임 요소 추가

## 🎨 배경 제거 (Canva Pro 기능)

### 자동 배경 제거

1. 캐릭터 이미지 선택
2. 상단 메뉴에서 **"배경 지우기"** 클릭
3. 1~2초 대기
4. 투명 배경 완성!

**장점**: DALL-E와 달리 추가 도구 불필요!

## 📥 다운로드 방법

### 개별 다운로드

1. 각 페이지 선택
2. 우측 상단 **"공유"** → **"다운로드"**
3. 파일 형식: **PNG**
4. ✅ **"투명 배경"** 체크
5. 파일명: `character_[emotion].png`
6. **"다운로드"** 클릭

### 일괄 다운로드 (추천)

1. **"공유"** → **"다운로드"**
2. 파일 형식: **PNG**
3. ✅ **"투명 배경"** 체크
4. **"모든 페이지"** 선택
5. **"다운로드"** 클릭
6. ZIP 파일로 다운로드
7. 압축 해제 후 파일명 수정

## 💡 Canva 작업 팁

### 일관성 유지
1. **색상 팔레트 저장**: 첫 캐릭터 완성 후 색상을 "브랜드 키트"에 저장
2. **페이지 복제 활용**: 새로 만들지 말고 복제해서 수정
3. **요소 잠금**: 변경하지 않을 부분(몸통 등)은 잠금

### 빠른 수정
- **Ctrl+D**: 페이지 복제
- **Ctrl+G**: 그룹화
- **Shift+드래그**: 비율 유지하며 크기 조절
- **Alt+드래그**: 복사

### 품질 향상
- 이미지 선택 → 상단 **"효과"** → **"그림자"** (선택사항)
- 테두리 추가로 명확성 향상 (선택사항)

## 🆘 문제 해결

### Q1: Magic Media가 안 보여요
**A**: Canva Pro 구독 확인. 무료 계정은 제한적.

### Q2: AI가 이상한 캐릭터를 만들어요
**A**: 
- 프롬프트를 더 구체적으로 수정
- "flat design", "simple", "children's app" 강조
- 여러 번 생성해서 좋은 것 선택

### Q3: 5개 캐릭터가 다 달라 보여요
**A**: 
- 첫 캐릭터 이미지를 PNG로 다운로드
- 2번째부터는 그 이미지를 캔버스에 올리고 참고
- 프롬프트에 "same style as previous" 추가

### Q4: 배경 제거가 완벽하지 않아요
**A**: 
- **"배경 지우기"** 재적용
- 또는 **"편집"** → 브러시로 수동 제거

### Q5: 파일 크기가 너무 커요
**A**: 
- 다운로드 시 **"파일 압축"** 옵션 체크
- 또는 TinyPNG (https://tinypng.com/) 사용
