# 📦 WP 1.6: 검사 모듈 C - 인지 제어 (Executive Functions)

마이크 녹음과 정밀 타이밍 측정이 필요한 고급 영역입니다. 수동 채점(작업 기억)과 자동 채점(주의 집중)이 혼합됩니다.

## 🧠 시나리오 1: 작업 기억 능력 (Working Memory)

- [ ] **S 1.6.1**: 숫자 따라 말하기 (순방향)
    - 숫자 시퀀스(예: "2-5-9") 음성 재생
    - 재생 완료 후 마이크 아이콘 활성화 → 아동이 따라 말함
    - 녹음 파일 저장 → 채점자 모드에서 수동 채점
    - 난이도: 2자리 → 3자리 → 4자리 → 5자리

- [ ] **S 1.6.2**: 숫자 거꾸로 말하기 (역방향)
    - "3-7"을 듣고 "7-3"으로 거꾸로 말하기
    - 녹음 저장 → 수동 채점
    - 순방향보다 짧은 자릿수로 시작 (2자리부터)

- [ ] **S 1.6.3**: 단어 따라 말하기
    - 단어 시퀀스(예: "사과-책상-구름") 재생 → 따라 말하기
    - 녹음 저장 → 수동 채점

- [ ] **S 1.6.4**: 단어 거꾸로 말하기
    - "나무-바다"를 듣고 "바다-나무"로 말하기
    - 녹음 저장 → 수동 채점

## 🎯 시나리오 2: 주의 집중 능력 (Attention / Go-No-Go)

- [ ] **S 1.6.5**: Go/No-Go 기본
    - 화면에 동물이 무작위로 등장 (1~2초 간격)
    - **Go**: 토끼가 나오면 빠르게 터치
    - **No-Go**: 늑대가 나오면 터치하지 않기
    - 측정: 정반응률, 오반응률, 반응 시간(ms)

- [ ] **S 1.6.6**: Go/No-Go 청각 버전
    - 화면은 고정, 소리로 판단
    - **Go**: '딩' 소리 → 터치
    - **No-Go**: '뿡' 소리 → 터치 금지
    - 시각 간섭 없이 순수 청각 주의력 측정

- [ ] **S 1.6.7**: 지속적 주의력 (Continuous Performance)
    - 30초~1분간 다양한 도형이 빠르게 지나감
    - 특정 도형(예: 빨간 별)이 나올 때만 터치
    - 측정: 집중 유지 시간, 후반부 정확도 저하 여부
