# 📦 WP 2.6: 학습 관리 시스템 (Learning Management)

일일 학습 제한, 진도 관리, 맞춤형 추천 시스템입니다.

---

## 💡 비개발자를 위한 전체 설명

**뭘 만드나요?**
아이의 학습 시간을 관리하고, 진도를 추적하고, 맞춤형 학습을 추천하는 "학습 매니저"입니다.

**왜 필요한가요?**
- 아이가 너무 오래 하면 피곤합니다 (하루 15분 제한)
- 어떤 부분을 배웠고, 무엇이 남았는지 보여줘야 합니다
- 검사에서 약했던 부분을 집중적으로 연습시켜야 합니다

---

## ⏱️ 시나리오 1: 일일 학습 제한

### S 2.6.1: 15분 타이머 시스템

**📋 기술적 요구사항:**
- 학습 시작 시 세션 타이머 가동
- 남은 시간 시각적 표시 (원형 프로그레스 등)
- 5분/1분 남았을 때 부드러운 알림

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
1. 학습을 시작하면 15분 타이머가 작동합니다
2. 화면 구석에 남은 시간이 표시됩니다 (예: ⏱️ 12:30)
3. 5분 남았을 때: "조금만 더 하면 끝이야!"
4. 1분 남았을 때: "마무리하자!"
5. 15분 도달: "오늘은 여기까지!"

**왜 15분인가요?**
- 만 6세 아이의 집중력은 약 15~20분입니다
- 너무 길면 피곤하고, 너무 짧으면 효과가 없습니다
- 부모 설정에서 10~30분으로 조절 가능합니다

---

### S 2.6.2: 학습 종료 처리

**📋 기술적 요구사항:**
- 15분 도달 시 "오늘은 여기까지! 내일 또 만나" 화면
- 현재 진행 상황 자동 저장
- 부모 설정으로 시간 조정 가능 (10분~30분)

**💡 비개발자를 위한 설명:**

**어떻게 끝나나요?**
- 캐릭터가 "오늘 정말 잘했어! 내일 또 만나!" 하고 손을 흔듭니다
- 오늘 얼마나 했는지 보여줍니다 (예: 20문제 완료, 15개 정답!)
- 자동으로 저장되어 내일 이어서 할 수 있습니다

---

## 🗺️ 시나리오 2: 진도 관리

### S 2.6.3: 단계별 잠금 해제

**📋 기술적 요구사항:**
- 1단계 학습 완료(일정 점수 이상) 시 2단계 잠금 해제
- 잠긴 단계는 자물쇠 아이콘 + "조금만 더 연습하면 열려요"
- 해제 시 축하 애니메이션

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
- 처음에는 1단계만 열려있습니다
- 1단계를 80% 이상 맞히면 2단계가 열립니다
- 자물쇠가 "딸깍!" 하고 열리는 애니메이션
- "새로운 모험이 열렸어!" 메시지

**왜 이렇게 하나요?**
- 아이가 성취감을 느낄 수 있습니다
- 너무 어려운 단계로 건너뛰는 것을 방지합니다
- 단계별로 차근차근 배울 수 있습니다

---

### S 2.6.4: 학습 진도 맵

**📋 기술적 요구사항:**
- 각 단계를 지도/여정 형태로 시각화
- 현재 위치, 완료한 스테이지, 다음 목표 표시
- 캐릭터가 여행하는 스토리텔링

**💡 비개발자를 위한 설명:**

**뭘 보여주나요?**
학습 진도를 "보물 찾기 지도"처럼 보여줍니다!

**화면 예시:**
```
🏰 [성] ────── 🌲 [숲] ────── 🏔️ [산]
      완료✓        현재위치        잠금🔒
```

- 이미 한 곳: 깃발이 꽂혀있음 ✓
- 지금 하는 곳: 캐릭터가 서있음 👦
- 아직 안 한 곳: 자물쇠가 잠겨있음 🔒

**왜 재미있을까요?**
- 게임이 아니라 "모험"처럼 느껴집니다
- "다음엔 어디 갈까?" 기대감을 줍니다

---

## 🔁 시나리오 3: 반복 학습

### S 2.6.5: 오답 문제 재출제

**📋 기술적 요구사항:**
- 틀린 문제는 변형(다른 보기, 다른 순서)하여 다시 출제
- 3회 연속 정답 시 해당 문제 완전 학습으로 처리

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
- 아이가 틀린 문제를 기억해둡니다
- 나중에 다시 나오지만, 조금 다르게 나옵니다
- 예: "사과-사탕 찾기" 틀림 → 나중에 "소나무-소방차 찾기"로 변형
- 3번 연속 맞히면 "완전히 배웠다"고 판단

**왜 필요한가요?**
- 한 번 틀렸다고 포기하지 않습니다
- 여러 번 연습할 기회를 줍니다
- 진짜 이해할 때까지 돕습니다

---

### S 2.6.6: 복습 세션 자동 생성

**📋 기술적 요구사항:**
- 일정 기간 후 이전 학습 내용 복습 문제 제공
- 에빙하우스 망각 곡선 기반 복습 주기

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
- 배운 지 1일 후 → 복습 문제 나옴
- 배운 지 3일 후 → 다시 복습
- 배운 지 7일 후 → 마지막 복습

**왜 필요한가요?**
- 사람은 배운 것을 잊어버립니다
- 적절한 시기에 복습하면 오래 기억합니다
- "망각 곡선" 이론에 기반한 과학적 방법입니다

---

## 🎯 시나리오 4: 맞춤형 추천

### S 2.6.7: 검사 결과 연동 추천

**📋 기술적 요구사항:**
- Milestone 1 검사에서 🟡/🔴 영역의 학습 콘텐츠 우선 추천
- "음절 쪼개기가 약해요 → [음절 블록 쪼개기] 게임 추천!"

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
1. Milestone 1 검사 결과를 분석합니다
2. 약한 영역(🟡🔴)을 찾습니다
3. 그 영역의 학습 게임을 우선 추천합니다

**실제 예시:**
```
검사 결과: 음절 쪼개기 50점 (🟡)
     ↓
앱 추천: "음절 블록 쪼개기 게임을 해보세요!"
     ↓
아이가 게임을 하면서 연습
     ↓
재검사: 음절 쪼개기 80점으로 향상! (🟢)
```

---

### S 2.6.8: 오늘의 학습 자동 구성

**📋 기술적 요구사항:**
- 약점 영역 + 복습 + 새 내용을 조합한 15분 세션 자동 생성

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
매일 "오늘의 학습"을 자동으로 만들어줍니다:
- 40% - 약한 부분 집중 연습
- 30% - 지난번 배운 것 복습
- 30% - 새로운 내용

**실제 예시:**
```
오늘의 학습 (15분)
├── 음절 쪼개기 게임 6분 (약점)
├── 소리 찾기 복습 4분 (복습)
└── 음절 합치기 5분 (신규)
```

**왜 좋은가요?**
- 부모가 고민하지 않아도 됩니다
- 앱이 알아서 최적의 학습을 구성해줍니다
- 균형 잡힌 학습이 가능합니다
