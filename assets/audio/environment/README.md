# 오디오 파일 안내

## 필요한 파일

다음 오디오 파일들을 이 폴더에 추가해야 합니다:

- `rain.mp3` - 빗소리
- `car.mp3` - 자동차 소리  
- `door.mp3` - 문 소리
- `water.mp3` - 물 소리
- `bird.mp3` - 새 소리
- `clock.mp3` - 시계 소리

## 무료 다운로드 사이트

### 1. Orange Free Sounds (추천)
- URL: https://orangefreesounds.com/raining-sound/
- 라이선스: CC-BY 4.0 (상업적 이용 가능, 출처 표시 필요)

### 2. Chosic
- URL: https://www.chosic.com/download-audio/28297/
- 라이선스: CC0 (출처 표시 불필요)

### 3. Freesound.org
- URL: https://freesound.org/
- 검색어: "rain", "car sound", "door knock" 등
- 라이선스: 다양 (각 파일 확인 필요)

## 빠른 테스트용

임시로 테스트하려면 위 사이트에서 `rain.mp3` 파일을 다운로드하여 이 폴더에 저장하세요.

파일을 추가한 후:
1. `flutter pub get` 실행
2. 앱 재시작 (Hot Restart)
3. 오디오 재생 테스트
