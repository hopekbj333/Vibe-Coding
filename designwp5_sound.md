## 🎶 카테고리 1: 효과음 (10개)

**기술 사양:**
- 형식: MP3 (128 kbps) 또는 AAC
- 샘플레이트: 44.1 kHz
- 채널: Mono
- 지속 시간: 0.5~2초
- 용량: < 100KB/파일

| 파일명 | 용도 | 지속 시간 | 분위기 |
|--------|------|----------|--------|
| `correct.mp3` | 정답 피드백 | 1.5초 | 밝고 긍정적 (딩동댕) |
| `incorrect.mp3` | 오답 피드백 | 1.5초 | 부드럽게 격려 (띠리링) |
| `button_click.mp3` | 버튼 클릭 | 0.3초 | 가볍고 경쾌 (톡) |
| `level_up.mp3` | 레벨업 | 2.0초 | 환호, 팡파르 |
| `encouragement.mp3` | 격려 효과음 | 1.0초 | 따뜻하고 응원 |
| `game_start.mp3` | 게임 시작 | 1.5초 | 기대감, 설렘 |
| `game_complete.mp3` | 게임 완료 | 2.0초 | 성취감, 축하 |
| `badge_unlock.mp3` | 배지 획득 | 1.5초 | 반짝, 성취 |
| `star_collect.mp3` | 별 획득 | 0.8초 | 반짝, 기쁨 |
| `whoosh.mp3` | 화면 전환 | 0.5초 | 부드러운 이동 |

**파일 경로**: `assets/audio/effects/`

## 🎤 카테고리 2: 음성 안내 (50개+)

## 🎼 카테고리 3: 배경음악 (3개)

**기술 사양:**
- 형식: MP3 (128 kbps)
- 샘플레이트: 44.1 kHz
- 채널: Stereo
- 지속 시간: 1~2분 (루프)
- 용량: < 2MB/파일

| 파일명 | 용도 | 분위기 | BPM |
|--------|------|--------|-----|
| `bgm_game.mp3` | 게임 중 | 밝고 경쾌, 집중력 유지 | 100~120 |
| `bgm_result.mp3` | 결과 화면 | 성취감, 축하 | 120~140 |
| `bgm_menu.mp3` | 메뉴 화면 | 편안하고 따뜻함 | 80~100 |

**파일 경로**: `assets/audio/music/`

### 음악 스타일
- 장르: 아동용 BGM (피아노, 실로폰 등 밝은 악기)
- 멜로디: 단순하고 반복적
- 볼륨: 배경에 깔리는 정도 (대화 방해 X)
- 루프: 자연스럽게 연결되도록

## 🎨 디자인 가이드

### 효과음 디자인 원칙
1. **명확성**: 정답/오답이 즉시 구분되는 소리
2. **친근함**: 부드럽고 귀여운 톤 (삐삐 같은 기계음 X)
3. **적절한 볼륨**: 놀라지 않을 정도
4. **짧음**: 0.5~2초 이내 (집중력 유지)

### 음성 안내 원칙
1. **속도**: 보통보다 15% 느리게
2. **발음**: 또박또박, 명확하게
3. **톤**: 따뜻하고 격려하는 느낌
4. **감정**: 과하지 않게, 자연스럽게

### 배경음악 원칙
1. **볼륨**: 낮게 (음성/효과음보다 낮음)
2. **반복**: 지루하지 않게, 변화 주기
3. **템포**: 빠르지 않게 (긴장감 유발 X)
4. **악기**: 아동 친화적 악기 (피아노, 실로폰, 우쿨렐레)

## 💡 디자이너를 위한 팁

### 효과음 제작
1. **정답 소리**: 밝고 긍정적 (예: 딩동댕, 반짝)
2. **오답 소리**: 부드럽고 격려 (예: 띠리링, 음~ - 단, 부정적이지 않게!)
3. **버튼 클릭**: 가볍고 경쾌 (예: 톡, 탁)
4. **레벨업**: 짧은 팡파르 (예: 따라라랑~)

### 음성 녹음
1. **스크립트 예시**:
   - "같은 소리를 찾아보세요" (inst_same_sound.mp3)
   - "잘했어요! 다음 문제로 가볼까요?" (enc_great.mp3)
2. **녹음 환경**: 조용한 스튜디오, 노이즈 최소화
3. **후반 작업**: 노멀라이즈, 노이즈 제거, EQ 조정
4. **테스트**: 실제 아동에게 들려주고 이해하는지 확인

### 배경음악 작곡
1. **악기 선택**: 피아노, 실로폰, 벨, 우쿨렐레 등 밝은 악기
2. **멜로디**: 단순하고 반복적, 귀에 쏙쏙
3. **템포**: 너무 빠르지 않게 (긴장감 유발 X)
4. **볼륨**: 음성/효과음보다 낮게 (-12dB 이상)
5. **루프**: 시작과 끝이 자연스럽게 연결

## 🔧 기술 사양 상세

### 녹음 설정
```
- 마이크: 콘덴서 마이크 (또는 스마트폰 고품질 녹음)
- 샘플레이트: 48 kHz (작업), 44.1 kHz (최종)
- 비트 뎁스: 24 bit (작업), 16 bit (최종)
- 포맷: WAV (작업), MP3 (최종)
```

### 후반 작업
```
1. 노이즈 제거 (Noise Reduction)
2. 노멀라이즈 (Normalize to -3dB)
3. EQ 조정 (중저음 강화)
4. 컴프레서 (Compressor, 다이나믹 조절)
5. MP3 변환 (128 kbps CBR)
```
