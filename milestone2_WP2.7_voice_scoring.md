# 📦 WP 2.7: 음성 채점 고도화 (Advanced Voice Scoring)

STT 연동을 통한 반자동 채점 시스템입니다.

---

## 💡 비개발자를 위한 전체 설명

**뭘 만드나요?**
아이의 녹음을 컴퓨터가 자동으로 글자로 바꿔서(음성 인식), 채점을 훨씬 빠르게 할 수 있게 하는 시스템입니다.

**왜 필요한가요?**
- 현재: 선생님이 녹음을 일일이 듣고 채점 → 시간이 오래 걸림
- 개선: 컴퓨터가 먼저 글자로 바꿔줌 → 선생님은 확인만 → 10배 빠름

**실제 예시:**
```
[현재 방식 - 느림]
선생님: 녹음 재생 → "7...3" 들음 → O 클릭
       (1문항 당 20초)

[개선 방식 - 빠름]
컴퓨터: 녹음 자동 인식 → "7-3" 텍스트 표시
선생님: 텍스트 보고 확인 → O 클릭
       (1문항 당 2초)
```

---

## 🎙️ 시나리오 1: STT 연동

### S 2.7.1: Speech-to-Text API 연동

**📋 기술적 요구사항:**
- Google Cloud Speech / Naver Clova Speech 연동
- 아동 음성 특화 모델 설정 (높은 피치, 불명확 발음 허용)

**💡 비개발자를 위한 설명:**

**뭘 했나요?**
구글이나 네이버의 "음성 인식 서비스"를 우리 앱에 연결했습니다.

**구체적으로:**
- **STT (Speech-to-Text)**: 말을 글자로 바꿔주는 기술
  - 아이가 "사과"라고 말함 → 컴퓨터가 "사과" 텍스트로 인식
  
- **아동 음성 특화**:
  - 아이들은 어른보다 목소리가 높습니다
  - 발음이 불명확할 수 있습니다
  - 이를 감안한 설정을 합니다

**실제 작동:**
```
아이: "바나나!" (높은 목소리, 약간 불명확)
  ↓
STT: "바나나" 인식 (90% 확신)
  ↓
앱: "바나나"로 표시 + 녹음 파일 첨부
```

---

### S 2.7.2: 실시간 음성 인식

**📋 기술적 요구사항:**
- 녹음과 동시에 STT 변환 시도
- 인식 결과를 채점 힌트로 활용

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
- 아이가 말하는 동안 실시간으로 인식합니다
- 학습 게임에서 즉각 피드백이 가능합니다

**실제 예시:**
```
[학습 게임 중]
아이: "나비" (말함)
  ↓ (즉시)
앱: "나비" 인식 → 정답! → "잘했어!" 효과음
```

## ⚡ 시나리오 2: 반자동 채점

### S 2.7.3: STT 결과 기반 자동 판정

**📋 기술적 요구사항:**
- STT 결과와 정답 비교 → 일치 시 자동 O
- 신뢰도 낮은 경우 "수동 확인 필요" 표시

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**

**Case 1: 명확한 정답**
```
정답: "7-3"
STT 인식: "7-3" (95% 확신)
  → 자동 O 처리 (선생님 확인 불필요)
```

**Case 2: 애매한 경우**
```
정답: "7-3"  
STT 인식: "7-2" (60% 확신)
  → ⚠️ 수동 확인 필요 → 선생님이 듣고 판단
```

**장점:**
- 명확한 70%는 자동 처리 → 시간 절약
- 애매한 30%만 선생님이 확인 → 정확도 유지

---

### S 2.7.4: 채점자 확인 간소화

**📋 기술적 요구사항:**
- STT 결과 미리보기 제공
- 채점자는 O/X만 빠르게 확정
- 채점 시간 대폭 단축

**💡 비개발자를 위한 설명:**

**화면 구성:**
```
[채점 화면]
문제: "숫자를 거꾸로 말하세요: 3-7"
정답: 7-3

STT 인식: "7-3" ✅ (신뢰도: 95%)
권장: 자동 정답 처리

[🎵 재생] [✅ 확정] [❌ 수정]
```

선생님은:
- 대부분 [✅ 확정]만 클릭 (2초)
- 의심스러울 때만 [🎵 재생]으로 확인

## 🗣️ 시나리오 3: 발음 피드백 (선택적)

### S 2.7.5: 발음 정확도 점수

**📋 기술적 요구사항:**
- 특정 음소의 발음 정확도 측정
- 학습 콘텐츠에서 발음 연습 시 즉각 피드백

**💡 비개발자를 위한 설명:**

**어떻게 작동하나요?**
- 아이가 "강아지"라고 말함
- STT가 "ㄱ" 소리를 정확하게 했는지 점수를 줌
- 점수: 0~100점

**실제 활용:**
```
아이: "강아지" (ㄱ 발음이 약간 부정확)
  ↓
STT: "강아지" 인식, 'ㄱ' 발음 점수 70점
  ↓
앱: "'ㄱ' 소리를 조금 더 세게 내봐!" 피드백
  ↓
아이: "강아지!" (다시 시도)
  ↓
STT: 'ㄱ' 발음 점수 90점
  ↓
앱: "잘했어!" ✨
```

**주의사항:**
- 너무 엄격하면 아이가 좌절합니다
- 70점 이상이면 OK!
- 격려 위주로 피드백
