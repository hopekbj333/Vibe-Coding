# 🗺️ 프로젝트 재설계 로드맵

**작성일:** 2025-12-08  
**프로젝트:** Literacy Assessment - Adaptive Learning System  
**새로운 방향:** 적응형 학습 경로 기반 문해력 준비도 검사 및 훈련  
**참고:** 현재 개발 기준은 35개 능력 체계 (음운인식능력 15개 + 음운처리능력 20개)

---

## 🎯 프로젝트 정체성 재정의

### **이전 (Old Identity)**
```
전통적 검사 방식
↓
5개 분야 50개 게임
↓
문항 제시 → 응답 → 채점
↓
결과 리포트
```

### **새로운 (New Identity)**
```
적응형 학습 경로
↓
50단계 위계 구조 (Tier 0~4 + Advanced)
↓
현재 수준 파악 → 다음 단계 제시
↓
AI 멘토 + 스토리 통합
↓
자기주도 학습
```

---

## 📊 새로운 50단계 구조 (확정)

### **전체 구조**

```
┌────────────────────────────────────────────────────┐
│              50-Step Learning Hierarchy             │
├────────────────────────────────────────────────────┤
│ Tier 0: Foundation          │ 10 steps │ 병렬 발달 │
│ Tier 1: Word Awareness      │  6 steps │ 선형 발달 │
│ Tier 2: Syllable Awareness  │  8 steps │ 선형 발달 │
│ Tier 3: Onset-Rime          │  8 steps │ 선형 발달 │
│ Tier 4: Phoneme Awareness   │  8 steps │ 선형 발달 │
│ Advanced Skills             │ 10 steps │ 병렬 발달 │
├────────────────────────────────────────────────────┤
│ Total                       │ 50 steps │           │
└────────────────────────────────────────────────────┘
```

### **학술적 근거**

- **Oregon Department of Education** (2024)
- **Newcastle University** Phonological Awareness Research
- **ASCD** Early Literacy Guidelines
- **Pubmed** Literacy Predictors Studies

## ⚠️ 주요 결정 사항 (당신의 승인 필요)

### **1. 스토리 개수**
- [ ] Option A: 1개 스토리 (표준화 우선)
- [ ] Option B: 2-3개 스토리 (다양성 + 재검사)
- [ ] Option C: 스토리 선택적 (일반 모드도 제공)

### **2. AI 멘토 수준**
- [ ] Level 1: 템플릿 기반 (하드코딩)
- [ ] Level 2: 제한적 LLM (검사 후 피드백만)
- [ ] Level 3: 풀 LLM (실시간 대화)

### **3. 기존 게임 처리**
- [ ] Option A: 전부 재구성 (시간 많이 소요)
- [ ] Option B: 일부만 수정하고 점진적 개선
- [ ] Option C: 새 시스템과 병행 (레거시 유지)

### **4. 개발 우선순위**
- [ ] Option A: Phase 2→3→4 순차 진행
- [ ] Option B: Phase 2+3 병행 (빠른 프로토타입)
- [ ] Option C: Phase 4 먼저 (콘텐츠 우선)
