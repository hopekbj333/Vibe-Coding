## 🗺️ Chapter 1: 소리 섬 (음운인식능력 15문항)

### 📍 섬 소개

**한글의 대사**:
```
"첫 번째 섬은 '소리 섬'이야!
여기서는 소리를 듣고 구별하는 실력을 테스트해볼게.
소리를 잘 듣고 구별할 수 있으면 한글을 배우는 게 훨씬 쉬워져.
준비됐어? 그럼 출발해볼까?"
```

**섬 특징**:
- 다양한 소리들이 살고 있는 섬
- 동물 소리, 환경음, 음절 소리 등이 공존
- 각 지역마다 다른 소리 퀴즈가 있음

---

### 🎯 Stage 1-1: 기초 청각 능력 (2문항)

**장면**: 소리 섬 입구 - 동물 마을

**한글의 대사**:
```
"우와! 동물 마을이야!
여기 사는 동물들이 소리를 내고 있어.
같은 소리를 내는 친구들을 찾아줄 수 있을까?"
```

**문항 1: 환경음 식별**
- **스토리 맥락**: "동물 마을에서 개, 고양이, 자동차 소리가 들려. 어떤 소리인지 맞춰볼까?"
- **능력**: 0.1 환경음 식별
- **게임**: 동물 소리 이야기 (`animal_sound_story_game_v2.dart`)
- **피드백 (정답)**: "맞아! 정확하게 구별했어! 👏"
- **피드백 (오답)**: "아쉽지만 괜찮아! 다시 한 번 들어볼까? 🎵"

**문항 2: 소리 크기/높이 변별**
- **스토리 맥락**: "이번엔 소리의 크기와 높이를 구별해볼게. 큰 소리와 작은 소리, 높은 소리와 낮은 소리를 찾아줘!"
- **능력**: 0.2 소리 크기/높이 변별
- **게임**: 음높이 구별, 소리 크기 비교
- **피드백 (정답)**: "훌륭해! 소리의 차이를 정확히 알아챘어! ✨"
- **피드백 (오답)**: "조금 어려웠나? 다시 들어보면서 차이를 느껴볼까? 💪"

```
"이제 '음절 숲'에 도착했어!
```

- **스토리 맥락**: "이번엔 단어를 음절로 나눠볼게. '가방'을 '가'와 '방'으로 나눌 수 있을까?"
- **능력**: 1.2 음절 분절
- **게임**: 음절 분리 (`syllable_split_game_v2.dart`)

- **스토리 맥락**: "반대로 음절을 합쳐서 단어를 만들어볼게. '가'와 '방'을 합치면 뭐가 될까?"
- **능력**: 1.3 음절 합성
- **게임**: 음절 합성 (`syllable_merge_game_v2.dart`)

**문항 6: 음절 변별**
- **스토리 맥락**: "같은 소리와 다른 소리를 구별해볼게. '가방'과 '가지'는 같은가요, 다른가요?"
- **능력**: 1.4 음절 변별
- **게임**: 같은 소리 찾기, 다른 소리 찾기
- **피드백 (정답)**: "정확해! 소리를 잘 구별했어! 👏"
- **피드백 (오답)**: "다시 들어보면서 차이를 느껴볼까? 🎵"

**문항 7: 음절 탈락**
- **스토리 맥락**: "이번엔 음절을 빼는 게임이야. '가방'에서 '가'를 빼면 뭐가 남을까?"
- **능력**: 1.5 음절 탈락
- **게임**: 각운 찾기 (`rhyme_game_v2.dart`)

## 📝 시나리오 작성 가이드

### 캐릭터 대사 원칙

1. **친근하고 격려하는 말투**
   - "와!", "대단해!", "훌륭해!" 등 긍정적 표현
   - 실패해도 "괜찮아!", "천천히 해볼까?" 등 격려

2. **아동 수준에 맞는 설명**
   - 어려운 용어 최소화
   - 구체적인 예시 제공
   - 단계별 설명

3. **스토리 맥락 유지**
   - 각 문항마다 스토리 맥락 제공
   - 장면 전환 자연스럽게
   - 보상과 피드백 스토리와 연계

### 피드백 메시지 원칙

1. **정답 피드백**
   - 구체적인 칭찬
   - 이모지 활용 (👏, ✨, 🎉 등)
   - 다음 단계로 자연스럽게 전환

2. **오답 피드백**
   - 격려 메시지
   - 힌트 제공
   - 재시도 기회 제공 (학습 모드)

3. **학습 중심**
   - 왜 정답인지 간단히 설명
   - 다음에 어떻게 하면 좋을지 제안
   - 실패를 학습 기회로 전환
